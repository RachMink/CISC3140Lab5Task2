PHP

Information

PHP which stands for Hypertext Preprocessor is a popular general-purpose scripting language that is especially suited to web development. PHP files can contain text, HTML, CSS, JavaScript, and PHP code. PHP code is executed on the server, and the result is returned to the browser as plain HTML. PHP files have extension ".php". They can generate dynamic page content, and can create, open, read, write, delete, and close files on the server.

Sample Syntax

<?php echo 'print this string'?>

*prints 'print this string' on a webpage

<?php echo(pi()); ?>

*prints returns 3.141592653589 on a webpage

<?php
$txt = "Hello world!";
$x = 5;
$y = 10.5;
?>

*creates three variables: $txt , $x and $y each assigned a different value
