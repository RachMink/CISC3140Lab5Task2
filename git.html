git/SVN

git

info

Git is a fast, scalable, distributed revision control system with an unusually rich command set that provides both high-level operations and full access to internals.

Sample Syntax

$ git commit filename -m message

*creates a new commit containing the contents of the file and the given log message describing the changes.

SVN

info

SVN stands for Subversion. Subversion is used for maintaining current and historical versions of projects. Subversion is an open source centralized version control system. It's licensed under Apache. It's also referred to as a software version and revisioning control system.

Sample Syntax

$ svn update

*brings changes from the repository into your working copy.
