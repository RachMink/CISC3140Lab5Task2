cURL/wGET

cURL

info

command line tool and library for transferring data with URLs (since 1998). Curl is used in command lines or scripts to transfer data.

Sample Syntax

$ curl -o thatpage.csv http://www.example.com/

*downloads files from example.com into thatpage.csv in you current directory

wGET

info

GNU Wget is a computer program that retrieves content from web servers. It supports downloading via HTTP, HTTPS, and FTP. Its features include recursive download and conversion of links for offline viewing of local HTML.

Sample Syntax

wget -c http://releases.ubuntu.com/ubuntu-18.04-live-server-amd64.iso

*if connection drops while downloading, instead of starting from scratch you can continue the previous download with -c option
