CSS

Information

Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language such as HTML. CSS is a cornerstone technology of the World Wide Web. It is designed to enable the separation of presentation and content, including layout, colors, and fonts. This separation can enable multiple web pages to share formatting by specifying the relevant CSS in a separate .css file which reduces complexity and repetition in the structural content.

Sample Syntax

.class {
color: red;
text-align: center;
}
*selects HTML elements with a specific class

#para1 {
text-align: center;
color: red;
}
*selects a specific HTML element using its id

p {
text-align: center;
color: red;
} *selects all <p> elements in the HTML file
